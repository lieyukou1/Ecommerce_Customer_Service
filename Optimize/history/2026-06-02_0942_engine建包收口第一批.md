# 2026-06-02 09:42 engine 建包收口第一批

## 本轮目标

把开始变得拥挤的 `engine/` 平铺目录收成子包结构，让后续重构不再靠一堆同级模块硬撑。

## 为什么现在做

前两轮虽然已经把大文件拆短了，但 `engine/` 目录开始出现新的问题：

- 模块数量快速增加
- 顶层目录缺少结构信号
- 后续继续拆分会让阅读成本反向上升

所以这轮优先做目录级收口，而不是继续平铺新增模块。

## 实际改动

1. 新建子包：
   - `engine/focus/`
   - `engine/runtime/`
   - `engine/turns/`
2. 迁移模块：
   - `focused_object_*` 与 `object_slot_handoff` -> `engine/focus/`
   - `runtime_directives` 与 `state_machine` -> `engine/runtime/`
   - `turn_decision_protocol` 与 `turn_models` -> `engine/turns/`
3. 修正 `dialogue_engine.py` 和相关模块 import
4. 修复一处建包后暴露出的循环导入：
   - 不再让子包 `__init__.py` 主动导入实现模块
   - 改为业务代码直接从具体模块导入
