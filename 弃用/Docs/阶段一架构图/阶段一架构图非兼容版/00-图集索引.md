# 阶段一架构图集索引（非兼容版）

## 这册看什么

这是阶段一架构图集的在线渲染版。

它和 Typora 兼容版的区别只有一个重点：这一套优先使用 Mermaid 更完整的表达能力，包括 `sequenceDiagram`、`classDiagram`、`stateDiagram-v2` 和更清晰的分组结构，用来在线查看、讲解和做设计对照。

它不负责：

- 适配老版本 Typora 的 Mermaid 限制
- 替代当前代码本身
- 把老师预期但本地未落地的内容伪装成“已实现”

## 标记规则

- `[已实现]`：当前正式项目代码里已存在
- `[占位]`：当前文件或类已存在，但方法还是空壳或最小骨架
- `[预留]`：老师架构中存在，当前仓库还没正式落地

## 推荐用法

- 想把整体快速讲给别人听：`01 + 02`
- 想讲清 infer 这一层：`04`
- 想讲清状态底盘：`07`
- 想看任务流怎么接进物业业务：`05 + 09 + 10`
