# 阶段一架构图集索引

## 定位

这是一套面向当前物业版项目的阶段一架构图集。

它的目标不是复述 `Docs/day01~day06` 的授课顺序，而是把已经搭起来的项目骨架按“整体 -> 分层 -> 轨道 -> 状态 -> 数据流 -> 映射”的方式重新组织，方便后续查阅、讲解和继续实现。

## Mermaid 兼容说明

为了适配当前 Typora 版本，本图集统一采用保守写法：

- 流程图优先使用 `graph TD` / `graph TB`
- 节点统一写成 `A["文本"]`
- 尽量不用复杂 `subgraph`、实验性语法和过长边标签
- 类、字段、参数类型默认采用“Mermaid 主图 + Markdown 表格”混合表达
- 如果后续某张时序图在 Typora 中显示不稳定，可直接复制 Mermaid 到外部工具渲染

## 已实现 / 架构预留 标记

- `[已实现]`：当前正式项目代码里已经存在
